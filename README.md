# alpine-demo
 a simple apline js demo


## install instructions
simply clone the repo then run
```
npm install
```
then run
```
npm start
```
then navigate to port 3000 in your browser. 


## acme landing page
clone the acme branch to see a running aplication then run
```
npm install
npm start
```
then navigate to port 3000 in your browser.
